[행추가형]양식은 작성되어진 양식을 입력시에 한행씩 추가하면서 입력되어집니다.

또한 취합시에도 입력된 자료는 형태 그대로 보여지므로 필요한 항목은 모두 추가하여야 합니다.

관련양식: 직원명단, 주소록 등 입력자가 여러 건을 넣어야 할때 사용합니다.

입력하는 화면은 아래 미리보기와 같습니다.
ㆍ위 화면은 속성정의 단계에서 미리보기를 한 것입니다.
ㆍ[동별]의 선택란, [계]의 자동계산, [내국인, 재외국인, 외국인]의 입력란은 양식정의후 속성
정의 단계에서 지정한 것입니다.
ㆍ행추가형에서는 양식정의시 사용자가 [연번]열을 작성하지 않아도 자동으로 [연번]열을
추가하여 입력담당자가 입력한 자료에 연번을 부여하게 됩니다.
ㆍ실제 제출대상 부서로 배부된 문서에 자료입력시에는 미리보기에서 보여지는 기능 이외에
입력된 자료의 수정 / 삭제가 가능합니다.

취합된 자료를 보여주는 화면은 아래 미리보기와 같습니다.
ㆍ위 화면은 속성정의 단계에서 미리보기를 한 것입니다.
ㆍ[연번]은 입력담당자가 입력하지 않아도 자동으로 부여하게 됩니다.
ㆍ실제 제출대상 부서에서 제출된 문서의 취합자료는 미리보기에서 보여지는 기능 이외에 제출된
자료의 제출부서 / 담당 / 입력담당자를 선택하여 원하는 자료만 확인할 수 있습니다.
 
   
 
 
 
 
 
 
 

[고정양식형]양식은 작성되어진 양식을 입력시에 동일한 화면으로 행추가없이 입력하게 됩니다.

입력화면 기준에서 동일한 화면으로 표시 되며, 표의 변화가 없습니다.

또한 취합시에는 부서항목이 자동으로 추가되며, 부서별 요약정보를 편리하게 보실 수 있습니다.

수치는 자동 합산되므로 별도의 계산이 필요없습니다.

관련양식: 동별인구현황, 동별증명서발급현황 등에 사용하시게 됩니다.

입력하는 화면은 아래 미리보기와 같습니다
ㆍ위 화면은 속성정의 단계에서 미리보기를 한 것입니다.
ㆍ[계]의 자동계산, [인구수, 세대수, 남, 여]의 입력란은 양식정의 후 속성정의 단계에서
지정한 것입니다.
ㆍ고정양식형에서는 양식정의시 사용자가 [부서]열을 작성하지 않아도 자동으로 [부서]열을
추가하여 입력담당자의 소속부서를 부여하게 됩니다.
ㆍ실제 제출대상 부서로 배부된 문서에 자료입력시에는 미리보기에서 보여지는 기능 이외에
입력된 자료의 수정이 가능합니다.

취합된 자료를 보여주는 화면은 아래 미리보기와 같습니다.

ㆍ위 화면은 속성정의 단계에서 미리보기를 한 것입니다.
ㆍ[부서]는 입력담장자가 입력하지 않아도 입력담당자의 소속부서를 자동으로 부여하게 됩니다.
ㆍ각 부서의 [소계]와 [총계]는 제출된 문서의 취합시 자동으로 표시하게 됩니다.
ㆍ실제 제출대상 부서에서 제출된 문서의 취합자료는 미리보기에서 보여지는 기능 이외에 제출된
자료의 제출부서 / 담당 / 입력담당자를 선택하여 원하는 자료만 확인할 수 있습니다.

 
   
 
 
 
 
 
 
 

[제본자료형]양식은 [행추가형]이나 [고정양식형]으로 취합하기 어려운 형식이나 각종 이미지나 파일을 취합하기 위한 용도로 사용합니다.

취합담당자가 양식예시파일을 제공할 수 있고, 관련된 파일을 제공할 수도 있습니다.

입력담당자는 이미지나 파일을 형식에 상관없이 등록하게 됩니다.

ㆍ위 화면은 입력하는 담당자의 화면입니다.
ㆍ취합담당자가 제공한 양식파일을 다운로드하여 내용작성후 다시 분류와 제목을 입력하고,
파일을 선택하여 등록합니다.


ㆍ위 화면은 취합이 완료된 화면입니다.
ㆍ입력부서, 담당계, 담당자를 변경하면 해당조건으로 취합된 내용만 보이게 됩니다.
ㆍHWP문서만 취합이 되었다면, 제본자료 생성버턴을 눌러 취합된 첨부파일을
전부 하나의 파일로 병합하여 볼 수 있습니다.

 
   
 
 
 
 
 
 
 

[행추가형]양식은 입력시에 한행씩 추가하여 여러건을 입력할 수 있고, [고정양식형]은 정해진 표에 맞추어 입력자에 해당하는 항목만 입력할 수 있습니다.

취합시에는 [행추가형]양식은 입력되어진 자료 그대로하여 연번만 추가되어 목록형식으로 보여지며, [고정양식형]양식은 입력자의 부서가 추가되어 요약된 형태로 보여지게 됩니다.

요약하면 입력할때는 행추가 되냐 안되냐에 따라 차이가 있으며, 취합시에는 부서별로 요약해서 보여주는냐에 따른 차이가 있습니다.
